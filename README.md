# Maven-SSM
* 一个基于maven的Spring+SpringMVC+Mybatis的项目基本配置。包含了mybatis的逆向工程。项目里包含一个数据库（db_onlineqa）和一张表（users）。   
* 数据库请自建，数据库名为root，密码为root。如需更改，请定位到**OnlineQA/src/main/resources/jdbc.properties**中修改。
